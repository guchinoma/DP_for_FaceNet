# DP_for_FaceNet
Producing adversarial examples for FaceNet

# Requirements
Tensorflow 0.12.1
Python 2.7.13
numpy 1.11.3 or higher
six 1.0 or higher

# Preparation
Make "data" directory in this directory. Put the [LFW dataset](http://vis-www.cs.umass.edu/lfw/) in that directory

# How to use
python lfw_driver.py
